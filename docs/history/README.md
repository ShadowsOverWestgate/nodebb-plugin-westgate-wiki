# Historical design documents

These contracts and implementation plans describe **completed** work and the
design of subsystems as they were being built. They reference modules that no
longer exist (the slug-based path engine, `wiki-path-migration.js`,
`wiki-namespace-main-pages.js`, `wiki-canonical-path-adapter.js`) and are kept
for archaeology only. For the current architecture, read
[../../ARCHITECTURE.md](../../ARCHITECTURE.md).
